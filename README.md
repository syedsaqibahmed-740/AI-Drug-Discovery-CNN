# AI-Drug-Discovery-CNN
AI Drug Discovery using Convolutional Neural Networks (CNN)

This project uses Deep Learning techniques to assist in Drug Discovery. 
The model is built using CNN architecture to analyze molecular datasets 
and predict potential drug compounds.

Tools Used:
- Python
- TensorFlow / Keras
- Pandas
- NumPy
- Google Colab

Steps:
1. Data Collection
2. Data Preprocessing
3. Model Building
4. Training
5. Evaluation

This project is part of my AI/ML learning and internship preparation.
